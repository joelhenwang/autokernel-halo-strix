## Phase A — SPECTRA-post + μP-partial implementation (3 days)

Goal: land SPECTRA post-clipping and μP (init + LR scaling, no 1/d_head) behind flags, default OFF. All unit-tested.

### Task A.1 — SPECTRA module skeleton

**File:** `halo_training/spectra.py` (NEW)

**Change:** Create module with `apply_post_clip(update, clip_norm, ns_iterations)` function. Reuses the Newton-Schulz implementation from `halo_training/muon.py` (import the existing NS kernel rather than duplicating).

**Exit criterion:**
```python
from halo_training.spectra import apply_post_clip
import torch
M = torch.randn(64, 128)
clipped = apply_post_clip(M, clip_norm=1.0, ns_iterations=5)
assert clipped.shape == M.shape
# spectral norm bounded
u, s, v = torch.svd(clipped)
assert s[0].item() <= 1.01  # allow small NS approximation error
```

**Test:** `tests/test_spectra.py::test_post_clip_bounded`

**Rollback:** Delete `halo_training/spectra.py`; no other state change.

### Task A.2 — Integrate SPECTRA post into NorMuon

**Change:** Add optional `spectra_post=False` constructor arg + `spectra_clip_norm=1.0` + `spectra_ns_iter=5`. In `step()`, after computing update per param, optionally apply `apply_post_clip` before assigning.

### Task A.4 — Wire μP into `build_optimizer`

**File:** `halo_training/optimizer.py`

**Change:** Add `use_mup` kwarg to `build_optimizer`. When True, call `build_mup_param_groups` instead of the Sprint 1 2-way splitter. Preserve existing behavior when `use_mup=False`.

```python
def build_optimizer(model, base_lr, optimizer="adamw", lr_2d=None, lr_1d=None,
                   use_mup=False, mup_base_width=256, **kwargs):
    if use_mup:
        groups = build_mup_param_groups(model, lr_2d or 0.0235, d_base=mup_base_width)
    else:
        group_2d, group_1d = split_params_2d_vs_1d(model)
        groups = [
            {"params": [p for _, p in group_2d], ...},
            {"params": [p for _, p in group_1d], ...},
        ]
    # ... build optimizer from groups ...
```

**Exit criterion:** 
- `build_optimizer(model, ..., use_mup=True)` returns optimizer with 3 param groups
- `build_optimizer(model, ..., use_mup=False)` matches Sprint 1 behavior exactly
- 1-step training with each config produces finite gradients

**Test:** `tests/test_optimizer_build.py::test_mup_optimizer_has_3_groups`

**Rollback:** Keep `use_mup=False` default; no behavior change when flag absent.

## Phase B — Pre-ablation probes (~2 hrs compute)

### Task B.1 — SPECTRA clip_norm sweep orchestrator

**File:** `scripts/sweep_spectra_clip.py` (NEW)

**Change:** Python orchestrator that launches 3 DDP training runs with `--spectra-post --spectra-clip-norm {0.5, 1.0, 2.0}`, 200 steps each. Parses `rank0.log` for steady-state loss (steps 100-200 mean). Outputs winner.

```python
import subprocess, os

CLIP_NORMS = [0.5, 1.0, 2.0]
MAX_STEPS = 200
for clip in CLIP_NORMS:
    run_dir = f"checkpoints/sprint1.5-spectra-clip-{clip}"
    env = os.environ.copy()
    env["EXTRA_FLAGS"] = (
        f"--max-steps {MAX_STEPS} --spectra-post "
        f"--spectra-clip-norm {clip} --log-interval 25"
    )
    env["CKPT_DIR"] = run_dir
    env["TORCH_COMPILE_MODE"] = "max-autotune-no-cudagraphs"
    subprocess.run(["bash", "scripts/launch_ddp.sh"], env=env, check=True)

# Parse each rank0.log → pick lowest-loss winner
```

**Exit criterion:** All 3 runs complete; orchestrator prints winning clip_norm value.

**Rollback:** Delete orchestrator; manual runs if needed.

### Task C.2 — Run C2 (μP-partial alone)

**Command:**
```bash
EXTRA_FLAGS='--mup --mup-base-width 256 \
             --no-spectra-post --no-spectra-pre --no-mup-attn \
             --intra-doc-mask --value-residuals --head-gating --normuon \
             --lr-2d <B2_WINNER_TRANSFERRED> --lr-1d 0.007'
```

`B2_WINNER_TRANSFERRED`: the B2 winner LR is for 30M (d=512, d_ratio=2). For 122M (d=768, d_ratio=3), μP's hidden-layer LR multiplier is 1/3 vs 30M's 1/2. To match, pass the same `--lr-2d` value — the μP rule internally scales per-layer.

**Exit criterion:** 2-epoch run completes.

**Gate:** C2 final loss ≤ Sprint 1 baseline × 0.99 (≥ 1% improvement).

### Task C.3 — Run C3 (combined)

```bash
EXTRA_FLAGS='--spectra-post --spectra-clip-norm <B1_WINNER> \
             --mup --mup-base-width 256 \
             --no-spectra-pre --no-mup-attn \
             --intra-doc-mask --value-residuals --head-gating --normuon \
             --lr-2d <B2_WINNER> --lr-1d 0.007'
```

**Exit criterion:** 2-epoch run completes.

**Gate (Phase C → D):**
- C1 ≥ 1% improvement over S1 baseline
- C2 ≥ 1% improvement over S1 baseline
- C3 ≥ 1% improvement over S1 baseline

If C1 fails: diagnose SPECTRA; may indicate clip_norm or Newton-Schulz issue.
If C2 fails: μP parametrization regressing; investigate init scaling.
If C3 < max(C1, C2): subadditivity acceptable; proceed if combined still ≥1%.
